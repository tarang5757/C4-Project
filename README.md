###  C4 Project

# Tech: React + Tailwind CSS + Vite


# Getting Started


# 1. Clone the repository

```bash
git clone https://github.com/tarang5757/C4-Project.git
cd frontend
```

# 2. npm install

```bash
npm install
```

# 3. Run the development server

```bash
npm run dev
```

```bash
npm run build
```
